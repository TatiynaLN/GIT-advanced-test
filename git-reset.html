git-reset

сбрасывает HEAD в указанное состояние

для отмены последнего не запушенного действия

git reset --hard 05bbd1b где 05bbd1b - номер коммита к которому откатиться, все последующие исчезнут

git reset --hard 713e664 где 713e664 - номер коммита который необходимо вернуть, перейдем в состояние данного коммита
