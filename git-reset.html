git-reset

сбрасывает HEAD в указанное состояние

для отмены последнего не запушенного действия

git reset --hard 05bbd1b где 05bbd1b - номер коммита к которому откатиться, все последующие исчезнут

git reset --hard 713e664 где 713e664 - номер коммита который необходимо вернуть, перейдем в состояние данного коммита

просмотр истории. git log --oneline

git reset --soft 0d613db где 0d613db - номер коммита к которому перенесся указатель HEAD на коммит 0d613db, но изменения, которые произошли в коммите 20e8ad9 не были уничтожены, остались в стайджинге гита и могут быть зафиксированы проверить git status измененные файлы в откаченом коммите отслеживаются
