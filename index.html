git switch и git checkout

Переключение ветки:
git checkout "branch"
git switch "branch"


git checkout -b "branch"
git switch -с "new-branch"
git switch --create "new-branch"
git branch "new-branch"
примечание: git branch создает ветку, не переходя в нее

Cоздание новой ветки от текущего коммита и указание слежения на ветку удаленного репозитория:

git switch -c <branch> --track <remote>/<branch>
git checkout -b <branch> --track <remote>/<branch>

Создание ветки без родительского коммита

git switch --orphan <new_branch>
